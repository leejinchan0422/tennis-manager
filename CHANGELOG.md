## 0.7.0 — 2026-07-16 (코트 예외 + 게스트 관리)
1. 한 성별만 참석한 날의 코트 예외 (PM 최종 결정 — 공식 예외조건으로 기록)

  - 여자만 참석 → C 여복 / D 여복
- 남녀가 모두 나온 날에는 기존 규칙 유지 (C = 남복, D = 여복 기본, 혼복은 D, 남자가 많으면 D에 남복 추가)

- 게스트 = 정규회원과 분리된 별도 목록 (`db.guests`). 회원 명단에 자동으로 들어가지 않음
- 이름 · 성별 · 그 날 참석 여부 · 지각 예정시간 · 조기 귀가 예정시간 · G 표시
- 지각/귀가 시간은 날짜별 저장(`session.times`)이며 **내부 운영정보** → 공개 화면 비노출
- 자동 대진 정상 포함 / 당일 랭킹 포함 / **누적 랭킹 제외**
- 정규회원 전환 기능 (과거 게스트 기록은 합치지 않음 — PM 미확정 사항)
- 대진 기록이 있는 게스트는 삭제 차단 (지난 대진표의 이름을 지키기 위해)
- 공개 화면(현장·회원 조회)에는 이름 옆 작은 `G` 딱지만 표시
