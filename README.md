# Tennis Club Manager

회원 · 참석 · 자동 대진 · 점수 · 랭킹을 **웹 하나로** 운영합니다.
서버 없음. 로그인 없음. **인터넷이 없어도 동작**합니다. 데이터는 쓰는 기기 안에만 저장됩니다.

**자동 대진은 100% 규칙 기반입니다. AI(ChatGPT / Claude / OpenAI / Anthropic API)는 한 줄도 들어가지 않습니다. (PROJECT_BIBLE §3)**

## 어디서 되나

특정 OS 전용 앱이 아니라 **반응형 웹**입니다. 같은 주소를 열면 됩니다.

| 기기 | 상태 |
|---|---|
| 태블릿 가로 (아이패드 · 안드로이드) | ★ 현장 운영 기준으로 우선 최적화 |
| 태블릿 세로 | 지원 |
| 스마트폰 (아이폰 · 안드로이드) | 지원 (한 줄 배치, 표는 좌우 스크롤) |
| Windows PC · Mac (크롬 · 엣지 · 사파리 · 파이어폭스) | 지원 (마우스 hover, 넓은 화면 3열) |

화면 크기 기준: `~599px` 폰 / `≥600px` 태블릿 세로 / `≥900px` **태블릿 가로(운영 기준)** / `≥1280px` PC

## 실행 방법

```bash
npm install
npm run dev        # 이 컴퓨터에서 보기
npm run dev:lan    # 같은 와이파이의 태블릿·폰에서 보기 (터미널의 Network 주소로 접속)
npm run build && npm run preview   # 배포본 확인 (설치·오프라인까지 실제로 동작)
```

## 홈 화면에 설치 (PWA)

- **안드로이드 · 크롬 · 엣지**: 화면 위쪽에 뜨는 **홈 화면에 설치** 버튼을 누릅니다.
- **아이폰 · 아이패드 사파리**: 공유 버튼 → **홈 화면에 추가**. (사파리는 설치 버튼을 못 띄워서 안내만 나옵니다)
- 설치하면 주소창 없이 앱처럼 뜨고, 한 번 연 뒤에는 인터넷이 끊겨도 열립니다.

> 설치·오프라인 동작은 **https 또는 localhost**에서만 됩니다. (브라우저 규칙)
> 사내망·와이파이 IP(`http://192.168...`)로 열면 화면은 정상이지만 설치는 안 됩니다.

## 데이터가 저장되는 곳

데이터는 **기기마다 따로** 저장됩니다 (브라우저 localStorage, 키 `tcm.db`). 서버가 없으므로
태블릿에서 입력한 내용이 폰에 자동으로 나타나지는 않습니다.

- 옮기기: 회원관리 → **백업 저장**(JSON 파일) → 다른 기기에서 **백업 불러오기**
- 여러 기기에서 동시에 같은 데이터를 보려면 서버·동기화가 필요합니다 (PROJECT_BIBLE 방법 B, MVP 이후)

## 화면 순서

대시보드 → 회원관리 → 참석관리 → 자동 대진 → 대진표·점수 → 당일 랭킹

처음이면 **대시보드 → 샘플 명단 14명 넣기**로 전체 흐름을 바로 눌러볼 수 있습니다.

## 폴더

```
public/
  manifest.webmanifest   설치 정보(이름·아이콘·색)
  sw.js                  서비스워커 — 오프라인에서 열리게 하는 캐시
  icons/                 홈 화면 아이콘 (안드로이드 · iOS · 마스커블)
src/
  lib/
    storage.js     기기에 저장/읽기, 백업 파일
    members.js     회원 규칙
    session.js     하루 운영(날짜별) 데이터 · 시간 계산
    scheduler.js   ★ 자동 대진 엔진 (규칙 기반)
    ranking.js     당일 랭킹 계산
    pwa.js         설치 버튼 · 서비스워커 등록
  features/        dashboard / members / attendance / schedule / ranking
  ui.css           전체 디자인 + 반응형 (웹폰트 없음 = 오프라인에서도 동일)
```

## 기존 프로젝트에 합치는 경우

새 프로젝트를 만들지 마십시오. 기존 React 프로젝트의 `src/`를 이 저장소의 `src/`로 교체하고,
`public/`과 `index.html`의 PWA 관련 태그를 옮기면 됩니다. (CRA면 진입점이 `src/index.js`이므로
그 파일에서 `App`을 import 하도록 한 줄만 고칩니다.)
